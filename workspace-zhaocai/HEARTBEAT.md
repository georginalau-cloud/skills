**有异动** → 直接发推送报告（飞书 open_id: ou_e47f62cad7fec339fba10ae6e1f5a3c9，accountId: zhaocai）

格式：
```
📡 持仓快扫 HH:MM

🏠 个人持仓：
🔴/🟢 [标的名] [代码] [涨跌幅] [当前价] | 今日盈亏 ¥[金额] | 总浮盈 ¥[金额]([百分比])

📦 场内ETF：
🔴/🟢 [ETF名] [代码] [涨跌幅] [当前价] | 今日盈亏 ¥[金额] | 总浮盈 ¥[金额]([百分比])

📁 基金穿透（影响基金净值）：
🔴/🟢 [标的名] [代码] [涨跌幅] ← [持有基金名] | 对你的影响 ≈ ¥[金额]
⚠️ [高重叠标的] [代码] [涨跌幅] ← [N只基金重叠] | 合计影响 ≈ ¥[金额]

💱 外汇/大宗：
- [品种] [涨跌幅] [当前价]

📊 大盘：
- [指数名] [涨跌幅] ([点位])

💰 今日持仓总盈亏：¥[个股+ETF今日盈亏合计]
💰 总浮盈：¥[个股+ETF总浮盈合计]
```

### 计算公式

**个股/ETF 今日盈亏**：
```
今日盈亏 = (当前价 - 昨收价) × 持仓股数
```

**个股/ETF 总浮盈**：
```
总浮盈 = (当前价 - 成本价) × 持仓股数
```
成本价和持仓股数从 `holdings.json` 读取。

**基金穿透对你的影响**：
```
影响金额 ≈ 个股涨跌% × 该股在基金中的持仓比例% × 你持有该基金的市值
你持有该基金的市值 = 你的份额 × 最新净值
```
- 持仓比例从 `fund_holdings.json` 读取（每周一更新）
- 你的份额从 `holdings.json` 的 funds 部分读取
- 最新净值用天天基金 API 获取（如获取失败用成本价估算）

**高重叠标的合计影响**：
```
合计影响 = Σ(该股在每只持有基金中的影响金额)
```

### 数据源

- 个股/ETF 实时价格：`skills/shared/price_fetcher.py`（4级 fallback）
- 基金净值：天天基金 API（`fundgz.1234567.com.cn`）
- 持仓数据：`holdings.json`
- 基金穿透数据：`skills/market-alert/references/fund_holdings.json`

### ⚠️ 数据源故障处理

如果 price_fetcher 全部失败（代理问题）：
1. 先尝试直连模式（不走代理）：设置 `proxies=None` 或 `NO_PROXY=*`
2. 如果直连也失败，输出：「⚠️ 行情数据获取失败，请检查网络。」
3. **禁止输出"代理问题仍在"之类的技术细节** — 用户不关心原因，只关心结果
4. 如果部分数据可用（如个股OK但基金失败），输出可用部分，标注缺失部分
